# Bill Templates

This folder is used to store all the different bill templates. They are categorized into the following prints:

- **3InchPrints**: Thermal receipts, typically 80mm or 58mm roll templates.
- **A4Prints**: Full-page invoice and bill layouts.
- **A5Prints**: Half-page invoice and bill layouts.
- **Catalogues**: Dynamic product catalogs or catalog-style billing sheets.

You can place the code for converted templates inside the respective directories.
